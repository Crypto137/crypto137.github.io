Download

You can download the latest nightly build of MHServerEmu here. These represent current work-in-progress code and may contain issues. The server emulator is still in early stages of development, so we do not provide any stable versions as of right now.

The source code is licensed under AGPLv3 and is available on GitHub.

MHServerEmu requires original game files to function. You will have to legally acquire and provide them yourself to use the emulator. If you have ever played the game on Steam, you should still be able to download the game in your Steam library.

The setup instructions can be found in our documentation. If you are having issues setting everything up, feel free to ask for help on our Discord server in the #setup-help channel.
